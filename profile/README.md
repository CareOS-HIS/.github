# 🏥 CareOS-HIS (Healthcare Information Solutions)

ยินดีต้อนรับสู่พื้นที่ส่วนกลางขององค์กร **CareOS-HIS** ศูนย์รวมนวัตกรรมและการพัฒนาระบบเทคโนโลยีสารสนเทศเพื่อการแพทย์และสาธารณสุข เราใช้สถาปัตยกรรมแบบ **Monorepo** เพื่อให้การจัดการโปรเจกต์กระชับและมีประสิทธิภาพสูงสุด

องค์กรของเราประกอบด้วยโปรเจกต์หลักดังต่อไปนี้ (คลิกเพื่อเข้าไปยัง Repository ของแต่ละโปรเจกต์):

## 🔒 กฎระเบียบของทีม (Team Guidelines)
- **ห้าม** อัปโหลดไฟล์ Database Dump (เช่น `.sql`) หรือข้อมูลจริงของผู้ป่วยขึ้น GitHub เด็ดขาด (กรุณาใช้ `.gitignore`)
- การแก้ไขโค้ดหลักต้องทำผ่านระบบ **Pull Request (PR)** เท่านั้น
- เอกสารคู่มือและการวิเคราะห์โครงสร้างฐานข้อมูล (Schema Mapping) จะอยู่ในโฟลเดอร์ `docs/` ของแต่ละ Repository
